```rst
Usage
*****

Quickstart
==========

.. code-block:: bash

    pipx install hermesbaby
    hermesbaby new MyNewProject
    cd MyNewProject
    hermesbaby preview
    # Start editing ./docs/index.rst ...
```

```rst
    cd SeeWhatsPossible

    # Investigate preview in browser and sources in editor
```
